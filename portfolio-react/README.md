# Muhammet Şahin Yıldırım - Modern React Portfolio

A beautiful, modern portfolio website built with React, Styled Components, Framer Motion, and other cutting-edge technologies.

![Portfolio Preview](screenshot.png)

## Features

- **Modern Design**: Beautiful UI with smooth animations and modern aesthetics
- **Responsive Layout**: Looks great on all devices - mobile, tablet, and desktop
- **Interactive Elements**: Interactive components with animations and transitions
- **Dynamic Content**: Easily updatable content sections
- **Performance Optimized**: Fast loading and smooth performance
- **Particle Background**: Interactive particle animations on the home page
- **Typing Animation**: Dynamic text typing effect
- **Scroll Animations**: Elements animate as you scroll through the page

## Tech Stack

- **React**: Core library for building the UI
- **Styled Components**: For component-based styling with CSS-in-JS
- **Framer Motion**: For smooth animations and transitions
- **React Router**: For page navigation
- **React Icons**: For beautiful icons
- **AOS (Animate on Scroll)**: For scroll-based animations
- **Typed.js**: For typing animation effect

## Getting Started

### Prerequisites

- Node.js (v14 or newer)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/portfolio-react.git
cd portfolio-react
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Start the development server:

```bash
npm start
# or
yarn start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Customization

### Personal Information

Edit the text content in the component files to add your own information:

- `src/components/sections/Home.js` - Hero section content
- `src/components/sections/About.js` - About section content
- `src/components/sections/Education.js` - Education history
- `src/components/sections/Skills.js` - Your skills
- `src/components/sections/Projects.js` - Portfolio projects
- `src/components/sections/Contact.js` - Contact information

### Colors & Styling

Main styling variables can be found in `src/styles/GlobalStyle.js`. You can change the color scheme, typography, and other style properties here.

### Images

Replace the profile image and project images with your own in the respective component files.

## Deployment

Build the app for production:

```bash
npm run build
# or
yarn build
```

This creates a `build` folder with all the production files that can be deployed to any static site hosting service like Netlify, Vercel, GitHub Pages, etc.
