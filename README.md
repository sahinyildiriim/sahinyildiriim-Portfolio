# Muhammet Şahin Yıldırım - Portfolio Website

Modern, responsive portfolio website based on Muhammet Şahin Yıldırım's LinkedIn profile.

## Features

- Responsive design that works on all devices
- Clean and modern interface
- Sections for showing skills, projects, education, and contact information
- Contact form for potential clients or employers
- Social media links

## Usage

1. Open `index.html` in your browser to view the website
2. Customize the content in `index.html` to add your own information
3. Modify the styles in `css/style.css` to change colors, fonts, or layout

## Customization

### Changing Profile Information

Edit the `index.html` file to update:

- Profile image
- Name and title
- About section
- Education details
- Skills and projects

### Changing Colors

The main color scheme can be modified in the `css/style.css` file by changing the root variables:

```css
:root {
  --primary-color: #007bff; /* Main color */
  --secondary-color: #6c757d; /* Secondary color */
  --dark-color: #343a40; /* Dark color for text and backgrounds */
  --light-color: #f8f9fa; /* Light color for backgrounds */
}
```

### Adding Projects

To add more projects, copy and paste the project card HTML structure in the projects section:

```html
<div class="project-card">
  <div class="project-info">
    <h3>Project Title</h3>
    <p>Project description goes here.</p>
    <div class="project-tags">
      <span>Technology 1</span>
      <span>Technology 2</span>
      <span>Technology 3</span>
    </div>
    <div class="project-links">
      <a href="#" target="_blank"><i class="fas fa-link"></i> Demo</a>
      <a href="#" target="_blank"><i class="fab fa-github"></i> GitHub</a>
    </div>
  </div>
</div>
```
